# ALX Project 0x02 Setup

## Next.js Project Setup and Basics

This project demonstrates a basic Next.js setup, including:

- Pages and routing (`/home`, `/about`, `/posts`, `/users`)
- Components (`Card`, `PostCard`, `UserCard`, `PostModal`, `Button`, `Header`)
- Fetching data from APIs (JSONPlaceholder)
- Using Tailwind CSS for styling
- Dynamic rendering of posts and users
- Reusable components and props usage
